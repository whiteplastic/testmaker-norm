Information
Cronjob-Script to process the survey-jobs can be found at portal/cron/cronjob.php

It is configured by default to limit sending to 300 mails per 10 minutes. If no jobs are present or limit is reached, the script finishes.

If you have no cronjobs available, you can manually start the script here.
Please note that this only works as long as the corresponding browser window is opened.
