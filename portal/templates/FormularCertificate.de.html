Bescheinigung erstellen

Sie können Ihre Bescheinigung an dieser Stelle mit einer Registrierung verbinden. Diese bietet u. a. folgende Vorteile:
  • Sie können Ihre Bescheinigung beliebig oft ausdrucken
  • Sie können Ihr Feedback jederzeit erneut einsehen
  • Falls Sie an mehreren Tests teilnehmen, können Sie auf alle über ein Konto zugreifen.
Mehr hierzu finden Sie in den aktuellen Datenschutzrichtlinien.

   

Registrierung als Benutzer

Bitte geben Sie die für die Registrierung notwendigen Daten an. Beachten Sie hierbei, dass Sie die Angaben (Name und Geburtsdatum) nach der Registrierung nicht mehr ändern können.

Einmaliges Erstellen der Bescheinigung

Bitte geben Sie Ihren Namen und Ihr Geburtsdatum ein. Die Bescheinigung kann nur einmal mit genau diesen Angaben erstellt werden, prüfen Sie daher, ob alle Eingaben korrekt sind.

Nach Erstellung der Bescheinigung werden keine personenbezogenen Daten im System gespeichert und es kann keine weitere Bescheinigung zu diesem SelfAssessment erstellt bzw. ausgedruckt werden.

    Vollständiger Name
    (Vor- und Nachname):

    Geburtstag:
    (TT.MM.JJJJ)

    Benutzername:

    Passwort:

    Passwort bestätigen:

    E-Mail:
    * Wenn Sie Ihre E-Mail-Adresse angeben, wird die Bescheinigung an diese verschickt.

Zur Durchführung der Registrierung werden Sie im Anschluss ausgeloggt und Ihre TAN ungültig. Vom System wird eine E-Mail an die oben angegebene Adresse verschickt. Bitte bestätigen Sie Ihre Registrierung innerhalb von 7 Tagen. Danach können Sie sich anmelden und die Bescheinigung abrufen.
